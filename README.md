# Simulated lamp for DomoticASW

## Build and Run

This project is built using .NET 9.0. To build and run the project, you need to have the .NET SDK installed on your machine.

You can download the .NET SDK from the official [.NET website](https://dotnet.microsoft.com/download).

To build and run the project, just as a test, use the following command:

```bash
dotnet run --project src/Lamp.App/Lamp.App.csproj
```

## Run Tests

To run the tests, use the following command:

```bash
dotnet test tests/Lamp.Core.Tests/Lamp.Core.Tests.csproj
```
